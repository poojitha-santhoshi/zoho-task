## Getting Started

## Prerequisites
Tools that we need to run this app:

- ***[Node.js](https://nodejs.org/en/)***
- ***[Node Package Manager](https://www.npmjs.com/get-npm)***
- ***[MongoDB](https://www.mongodb.com/try/download/community)***

## Installing
```
npm install
```

## To Run the App
```
node server.js
```

The server will start Running on
+ http://localhost:3000/
